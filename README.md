# Generative-AI-with-Langchain-Huggingface-LLMs
This repository is a structured and hands-on learning journey through modern Generative AI development, covering the complete path from Python programming and Natural Language Processing fundamentals to Large Language Models (LLMs), Retrieval-Augmented Generation (RAG), Multi-Agent Systems, Fine-Tuning, Cloud Deployment, and Production-Ready AI Applications.

The goal of this repository is to build a strong foundation in AI Engineering while implementing real-world projects using industry-standard tools such as LangChain, Hugging Face, OpenAI, Gemini, AWS Bedrock, SageMaker, NVIDIA NIM, and CrewAI.
